How to use

  1. Run the backend on your server (Oracle VM) on 0.0.0.0:8080.
  2. Run local-app/send_refs.py to register reference persons (embeddings are computed locally).
  3. Run local-app/sort_local.py to compute inbox embeddings locally and receive assignments from the backend. Files are moved/copied locally.
